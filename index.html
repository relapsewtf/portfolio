acache

acache

Windows ActivitiesCache DB Parser for DFIR

Overview

A single-file toolkit designed to parse Windows ActivitiesCache databases for digital forensics investigations. Built with efficiency and extensibility in mind.

Quick Start

python3 acache.py /path/to/activitiescache.db --exportjson out.json

Key Features

  • Robust parsing with sqlite3.Row for reliable data extraction
  • Streaming JSON parsing with low memory footprint
  • CSV and JSON export capabilities
  • Timeline visualization with matplotlib
  • Integrated file scanning with maliciousness scoring (0-100)
  • Generates interactive HTML reports with dark theme
  • Unit tests and CI/CD integration via GitHub Actions
  • VirusTotal integration for enhanced threat detection

Installation

python -m pip install -e .

Advanced Usage

Perform recursive file scanning for suspicious artifacts:

python3 acache.py /path/to/activitiescache.db --filescan /path/to/scan --filescan-recursive --vt-key YOUR_KEY
